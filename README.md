# Urban Vista

Demo: https://taniaomelko.github.io/urban-vista

## Project Overview

This project is a simple, responsive website built with HTML and CSS. It showcases fundamental web design techniques and provides a clean, visually appealing layout. The goal of this project is to practice and demonstrate HTML/CSS skills, including layout structuring, styling, and responsive design.

## Features

- **Responsive Design:** The website is fully responsive, ensuring it looks great on desktop, tablet, and mobile devices.
- **Modern CSS:** Utilizes modern CSS features to create flexible layouts.
- **Animations:**  Uses Animate.css for simple, eye-catching animations.
- **Cross-Browser Compatibility:** The project is compatible with major browsers, ensuring a consistent appearance across different platforms.

## Technologies Used

- **HTML:** For semantic and accessible web content.
- **CSS:** For layout, styling, and responsive design adjustments.
